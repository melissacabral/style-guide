What is this?

This is supplemental code for the course "Programming for Designers" by Melissa Cabral. It follows our lesson on Sass Language basics, and is a great example of how powerful Sass can be at saving us time and repetitive code.

Goals of this lesson:

You know how to design a visual style guide, and want to learn how to use CSS Preprocessing to make it quicker and easier to code a smart web style guide.

How to use this for your own style guide

  • Fork or clone the code from GitHub
  • Run sass --watch on the css folder
  • Edit css/partials/variables to tweak the colors, fonts and other configuration to your hearts' desire!
  • The compiled style.css file can be attached to your web page after picnic.css
<head>
